# Getting Started

Welcome to Zone UI Design System! This guide will help you get started with using our components.

## Installation

Install individual components using npm:

```bash
npm install @zone-ui/[component-name]
```

For example, to install the Button component:

```bash
npm install @zone-ui/button
```

## Usage

Import components directly from their packages:

```tsx
import { Button } from '@zone-ui/button';

// Use the component
<Button>Click me</Button>
```

## Development

To run the documentation locally:

```bash
npm run docs:dev
```

The documentation will be available at `http://localhost:3000`
